# Nawsad Hossen Portfolio

A polished, responsive single-page portfolio built from Nawsad Hossen's resume.

## Highlights
- Clean hero, experience timeline, education, skills, and contact sections
- Responsive header with mobile-friendly navigation
- Scroll-based hero photo to header avatar reveal
- Light theme toggle
- Print-friendly styling and resume download

## Files
- `index.html`
- `styles.css`
- `script.js`
- `Nawsad-Hossen-Headshot.webp`
- `Nawsad-Hossen-Resume.pdf`

## Local Preview
```bash
python3 -m http.server 8080
```
Open `http://localhost:8080` in your browser.
